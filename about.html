üì± Project Overview

The Net Worth Tracker is a modern web application designed to help individuals monitor and manage their personal assets in Philippine Pesos. Built with a beautiful iOS-inspired interface, this application provides an intuitive way to track your financial portfolio and calculate your total net worth in real-time.

üéØ Purpose & Goals

This project was developed as part of the Web Systems and Technology course at Mapua Malayan Digital College. Our primary objectives were to:

  • Create a user-friendly financial tracking application
  • Implement modern web design principles and best practices
  • Utilize local storage for data persistence
  • Provide real-time calculations and updates
  • Ensure responsive design for all devices
